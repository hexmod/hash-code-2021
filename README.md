# Hash Code 2021

Our 2021 Google Hash Code Entry

**Rank**

## Usage
* Save the competition data files in the data folder
* `pipenv install` - Install dependancies
* `pipenv run python main.py` - Run the helper script

## Dependancies
* Python version 3.7.2 or higher
* pipenv
